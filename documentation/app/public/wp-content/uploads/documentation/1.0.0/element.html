Element Asset Format

,
Key Type DescriptionValues
metadata Dict Contains metadata information about the asset. "metadata": { "identifier": "Namespace:id" }
└─ identifier ID The identifier of the asset. Namespace:id
attributes Dict Defines the attributes of the asset. "attributes": { "model": "Namespace:id", "animationController": "Namespace:id", "renderMaterial": "Namespace:id" }
└─ model ID Specifies the model of the asset. Namespace:id
└─ animationController ID Specifies the animation controller of the asset. Namespace:id
└─ renderMaterial ID Specifies the material of the asset. Namespace:id
└─ inventory Dict Defines the inventory configuration of the asset. "inventory": { "grid": { "slots": 10, "acceptedTypes": "any" }, "named": { "named1": { "acceptedTypes": "type1" }, "named2": { "modelSocket": "handSocket", "acceptedTypes": [ "type2", "example" ] } } }
└─ grid Dict Defines the grid-based inventory layout. "grid": { "slots": 10, "acceptedTypes": "any" }
└─ slots Int The number of slots available in the grid inventory. 0, 1, 2... num ≥ 0
└─ acceptedTypes Str Specifies the types of items accepted in the grid inventory. any, type1, type2, etc.
└─ acceptedTypes List Specifies the types of items accepted in the grid inventory. "acceptedTypes": [ "any", "type1", "type2" ]
└─ [INDEX] Str Specifies the types of items accepted in the grid inventory. any, type1, type2, etc.
└─ named Dict Defines the named inventory layout. "named": { "named1": { "acceptedTypes": "type1" }, "named2": { "modelSocket": "handSocket", "acceptedTypes": [ "type2", "example" ] } }
└─ {KEY} Dict Defines a named inventory slot. "mainHand": { "modelSocket": "mainHandSocket", "acceptedTypes": [ "type2", "example" ] }
└─ modelSocket Str Specifies the connection point between the item in the named inventory slot and the asset model. mainHandSocket, headSocket, chestSocket, customSocket, etc.
└─ acceptedTypes Str Specifies the types of items accepted in the named inventory slot. any, type1, type2, etc.
└─ acceptedTypes List Specifies the types of items accepted in the named inventory slot. "acceptedTypes": [ "any", "type1", "type2" ]
└─ [INDEX] Str Specifies the types of items accepted in the named inventory slot. any, type1, type2, etc.

Example Element JSON Layout:

{
    "metadata": {
        "identifier": "TranscendRuins:statue"
    },
    "attributes": {
        "model": "TranscendRuins:statue",
        "animationController": "TranscendRuins:statue",
        "renderMaterial": "TranscendRuins:opaque",
        "inventory": {
            "grid": {
                "slots": 20,
                "acceptedTypes": "any"
            },
            "named": {
                "mainHand": {
                    "modelSocket": "mainHandSocket",
                    "acceptedTypes": [
                        "weapon",
                        "tool"
                    ]
                },
                "offHand": {
                    "modelSocket": "offHandSocket",
                    "acceptedTypes": [
                        "consumable",
                        "explosive"
                    ]
                },
                "base": {
                    "modelSocket": "baseSocket",
                    "acceptedTypes": "any"
                }
            }
        }
    }
}
